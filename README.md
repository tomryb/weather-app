# 🌤️ SSR Weather App

A weather application built with React using Server-Side Rendering (SSR). It allows you to check the current weather in a selected city and compare it with other major cities in Poland.

---

## 📦 Technologies

- **React + React Router DOM**
- **Express** (for SSR)
- **Axios** (for fetching weather data)
- **WeatherAPI** (https://www.weatherapi.com)
- **Redux Toolkit** (optional, if used)
- **Plain CSS** (no Tailwind)

---

## 🚀 How to run the project locally

### 1. Clone the repository

```bash
git clone https://github.com/tomryb/weather-app.git
cd weather-app
```
### 2. Install dependencies
```bash
npm install
```
### 3. Build the project for production
```bash
npm run build
```
### 4. Run the application (port 3000)
```bash
npm run start:ssr
npm run dev
